I was first introduced to programming by a friend, and upon further research, I discovered that programming allows for opportunities to innovate and solve real-life problems using code. As it turns out, problem-solving is a passion of mine.

So I began learning HTML and CSS using W3Schools. I practiced by cloning existing web pages. Later, I delved into JavaScript using an Udemy course titled "The Complete JavaScript Course 2020(From Zero to Expert)" by Jonas Schmedtmann.

Alongside this, I enrolled in a frontend developer bootcamp at Altschool Africa in late 2020, which helped me to strengthen my knowledge of HTML, CSS, and JavaScript.

Learning JavaScript was so much fun for me because it made me feel alive and more of a developer. I practiced my JavaScript more through solving problems on HackerRank and CodeWar>

Additionally, I learned new web stacks such as React and Vue. Among them, I developed a particular interest in React and furthered my knowledge using online resources such as React Tutorial from "The Complete Web Development Bootcamp" by Dr.Angela Yu and React for Beginners, as well as other playlists on React by Codevolution on YouTube.



As an intern at Technicon NG , I was opportuned to work with senior developers who have raised the bar for what's expected in many web applications.
Through my online tutorials, bootcamp, and experience at Technicon NG, I had the opportunity to collaborate with a senior backend developer on a contract project, where I was solely responsible for developing the frontend of a copy trading website (evergreenffx.com ). This website is live with over a thousand active users.
While working on the Evergreenffx project, I was offered another contract-based opportunity that required me to redesign certain sections of a marketplace website ( kampusplaza). Juggling both projects tested my multitasking skills as I had deadlines to meet for both, but ultimately, I was able to deliver the Evergreenffx project on time. The delivery of the Kampusplaza project was delayed slightly.
In addition, I have completed some personal projects, a selection of which can be found in the project section.
Presently, I am in the process of outlining the design and feasibility of two new projects. One of them aims to address a security issue while the other is an ecommerce platform with unique features targeted at solving specific issues in my city. In the long term, I plan to expand the ecommerce project to other cities as well.
